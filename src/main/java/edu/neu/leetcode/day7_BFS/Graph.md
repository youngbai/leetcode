### Graph Data Structure
- Edge List
    > `[[a,b], [b,c], [a,c]]`
- Adjacent List
    > ```
    > 0 - [1,2]
    > 1 - [2,3]
    > 2 - [3]
    > 3 - []
    > 
    > List<List<Integer>> adjList
    > Map<Integer, List<Integer>> adjList
    > ```
- Matrix
    > ```
    >    0 | 1 | 2
    > 0  distance
    > 1  distance
    > 2  distance
    > ```
